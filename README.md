# Restaurant_Reservation_System

A full-stack web application that allows users to browse restaurants and make reservations seamlessly.

---

## 🚀 Live Demo
🔗 https://restaurant-reservation-system-murex.vercel.app/login

---

## 📌 Features

- 🔐 User Authentication (Login / Signup)
- 🍴 Browse Restaurants
- 📅 Book Table Reservations
- 📋 View Reservation Details
- ⚡ Fast and responsive UI
- 🔄 Client-side routing (React Router)
